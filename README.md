# Incident Response Labs

# Overview
This respository contains hands-on cybersecurity incident reponse investigations performed in my home lab enviroment.

The purpose of these labs is to simulate real-world security incidents and document the investigation process used by Security Operations Center analysts.

## Skills Practiced
- Incident detection
- Log analysis
- Threat investigation
- Indicator of Compromise (IOC) identification
- Security event documentation

## Tools Used
- SIEM (Wazuh / Splunk)
- Windows Event Viewer
- Linuix logs
- Wireshark

## Lab Investigations
Examples of incidents investigated:

- Brute force login attack
- Phishing email simulation
- Malware execution detection
- Suspicious PowerShell activity

  ## Investigations Format
  Each incident investigation include:
  
  1. Inident summary
  2. Indicators of compromise
  3. Log analysis
  4. Root cause
  5. Remediation steps
 
  ## Goal
  Develop practical incident response skills aligned with real SOC workflows.
